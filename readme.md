# Patal Input Method

팥알님의 한글 배열을 지원하는 입력기

## 신세벌식

- 초성은 오른손, 중성과 종성은 왼손
- 조합 상태에 따라 중성과 종성을 선택하는 오토마타
- 숫자키를 사용하지 않는 세벌식

## 특수 기능

- Esc 키를 입력하면 영문 상태로 전환
- ~~Spotlight 에 진입할 때 영문 상태로 전환~~
- ~~내장된 영문 자판 사용시 더 빠른 한/영 전환~~
